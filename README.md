
# Forms!

A lightweight approach for implementing form widget abstraction within PHP applications. 

## Installation

Via Composer:
```
{
    "repositories": [
        {
            "url": "https://github.com/McManning/Form.git",
            "type": "git"
        }
    ],
    "require": {
        "mcmanning/form": "dev-master"
    }
}
```
